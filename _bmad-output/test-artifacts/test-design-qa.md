## Execution Strategy

**Philosophy:** Tận dụng tối đa Playwright cho cả API và UI.

### Every PR: Unit & API (~10 min)
- Toàn bộ UNIT tests (.NET/xUnit).
- Toàn bộ API Integration tests (Playwright).
- **Trọng tâm:** P0, P1.

### Nightly: Full E2E & Perf (~30 min)
- Chạy toàn bộ kịch bản E2E (UI Playwright).
- Kiểm thử Concurrency (Stress test Conflict API).
- Kiểm thử kịch bản Edge Case (ECH).
